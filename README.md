# House-Price-Prediction
House Price Prediction is a Machine Learning project that predicts the estimated price of a house based on various features. This project uses Machine Learning algorithms to analyze housing data and provide accurate price predictions through an interactive user interface built with Streamlit.

**Features**
Predict house prices instantly,
User-friendly Streamlit web app,
Machine Learning based prediction,
Real-time input handling,
Clean and responsive UI

**Technologies Used**
Python,
Pandas,
NumPy,
Scikit-learn,
Streamlit

**Project Workflow**
Data Collection,
Data Preprocessing,
Model Training,
Model Evaluation,
Streamlit Deployment

**Input Parameters**
Area (sq ft),
Number of Bedrooms,
Bathrooms,
Parking,
Location,
Furnishing Status

**Output**
The model predicts the estimated house price based on the user inputs.

**Run the Project**
pip install -r requirements.txt,
streamlit run app.py
